# 🎯 Number Guessing Game

A Java console game — guess the randomly generated number before your score hits zero!

## ✨ Features

- **3 Difficulty Levels** — Easy (1–50), Medium (1–100), Hard (1–200)
- **Smart Scoring** — Fewer attempts = higher score (`max(100 - attempts × multiplier, 0)`)
- **Real-time Hints** — Too high 🔽 / Too low 🔼 feedback after each guess

## 🚀 Quick Start


> **Requires:** Java 8+
